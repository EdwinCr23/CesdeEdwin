Tipos de datos en MySQL

vachar

Se utiliza para almacenar cadenas de caracteres.Una cadena es una secuencia de caracteres. Se coloca entre comillas(simples): 'Hola'.
El tipo "vachar" define una cadena de longitud variable en la cual determinamos el máximo de caracteres. Puede guardar hasta 255 caracteres. Para almacenar cadenas de hasta 30 caracteres, definimos un campo tipo varchar (30).

int

Se usa para guardar valores numéricos enteros, de -2000000000 a 2000000000 aproximadamente.
Definimos campos de ete tipo cuando queremos representar, por ejemplo, cantidades.
